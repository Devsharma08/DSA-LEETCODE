## Deep Dive

### 🔹 Graphs & Nonlinear Structures
*Implemented high-frequency patterns often found in Google/Meta technical rounds.*
- **Representation:** Robust Adjacency List implementation for sparse graphs.
- **Traversal:** Recursive and Iterative DFS for deep path exploration.
- **Trie:** Implementation of Prefix Trees for $O(L)$ word lookups (where $L$ is word length).

### 🔹 The Binary Search Suite
*Focusing on $O(\log N)$ efficiency for large-scale data.*
- **Koko Eating Bananas:** Mastering the "Search Space" technique.
- **Median of Two Sorted Arrays:** Implementation of the partition-based $O(\log(\min(M,N)))$ approach.
- **Rotated Search:** Handling pivot-based searches in modified sorted arrays.

### 🔹 Advanced Linked Lists
- **Interweaving Technique:** Successfully solved "Copy List with Random Pointer" using constant space $O(1)$ instead of a Hash Map.
- **Sorting:** Divide and Conquer via Merge Sort on Linked List structures.
